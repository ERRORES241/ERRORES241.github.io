Приклади проєктів

  • Розумний дім

    Автоматизація освітлення, клімат-контроль, датчики руху та диму. Керування через Wi-Fi.

    ESP32 · STM32

  • Робототехніка

    Керування сервоприводами, зчитування енкодерів, обробка даних IMU для балансування.

    STM32F4 · RP2040 · ESP32

  • Медичні пристрої

    Пульсоксиметри, глюкометри, портативні ЕКГ. Вимагають низького споживання.

    nRF52 · STM32L4 · MSP430

  • Метеостанція

    Збір даних з датчиків BME280, SHT31. Відображення на OLED, передача в хмару.

    ESP32 · ATmega328P

  • Автомобільна електроніка

    Блоки керування двигуном, ABS, подушками безпеки. Протокол CAN, жорсткі вимоги.

    STM32 · Infineon TC · NXP S32

  • Промислова автоматика

    Програмовані логічні контролери (ПЛК), керування конвеєрами, моніторинг обладнання.

    STM32 · PIC32 · Cortex-M7

Коротка історія МК

  1. 1971

    Intel 4004 — перший мікропроцесор

    4-бітний процесор для калькуляторів. Заклав основу для майбутніх МК.

  2. 1974

    TMS1000 — перший мікроконтролер

    Texas Instruments випустила першу мікросхему з процесором, пам'яттю та I/O на одному кристалі.

  3. 1980

    Intel 8051 — галузевий стандарт

    8-бітний МК, який досі застосовується. Архітектура 8051 є базовою у навчанні.

  4. 1997

    Atmel AVR — RISC для мас

    Перший МК, оптимізований для мов високого рівня. Основа Arduino (2005).

  5. 2007

    ARM Cortex-M — домінування 32-біт

    Cortex-M3 від ARM стала основою для STM32 та десятків інших сімейств.

  6. 2016

    ESP32 — IoT для всіх

    Espressif вивела Wi-Fi+BT МК за $3. Демократизація підключених пристроїв.

  7. 2021

    RP2040 — відкрита платформа

    Raspberry Pi Foundation випустила власний МК за $0.7 з унікальним PIO-блоком.
